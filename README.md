# MEXC Demo Trader

Демо-приложение для практики трейдинга криптовалюты. Котировки и графики —
**настоящие** (с биржи MEXC, публичный API без ключей). Баланс и сделки —
**виртуальные**, хранятся только на вашем телефоне. Реальные деньги не
используются и не могут быть использованы — приложение не подключается к
торговым (приватным) эндпоинтам биржи.

## Возможности

- Поиск любой монеты, торгуемой на MEXC (пары к USDT)
- График цены (свечи) с выбором таймфрейма: 1m / 5m / 15m / 1h / 4h / 1d
- Живая цена через WebSocket
- Спот-покупка (long) и фьючерсы с плечом 1x–125x (long/short)
- Stop-Loss и Take-Profit
- Симуляция ликвидации при достижении расчётной цены ликвидации
- Демо-баланс 10 000 USDT с возможностью сброса
- История закрытых и ликвидированных позиций

## Как собрать APK (бесплатно, без своего компьютера)

Этот репозиторий уже содержит готовый workflow GitHub Actions
(`.github/workflows/build.yml`), который сам собирает APK при каждом пуше.

1. Залейте этот код в свой репозиторий на GitHub (через Codespaces,
   `git push`, или загрузку файлов через веб-интерфейс).
2. Перейдите во вкладку **Actions** репозитория.
3. Откройте последний запуск workflow **Build APK** (он стартует
   автоматически после пуша в `main`/`master`, либо запустите вручную через
   **Run workflow**).
4. Дождитесь зелёной галочки (сборка занимает 5–10 минут).
5. Внизу страницы запуска — раздел **Artifacts** → скачайте
   `app-release-apk`. Внутри будет файл `app-release.apk`.
6. Перенесите APK на Android-телефон и установите (потребуется разрешить
   установку из неизвестных источников в настройках).

## Важные технические детали

- **Без API-ключей**: используются только публичные эндпоинты
  `GET /api/v3/exchangeInfo`, `GET /api/v3/ticker/price`,
  `GET /api/v3/klines` и публичный WebSocket
  `wss://wbs-api.mexc.com/ws`. Эти эндпоинты не требуют авторизации и не
  дают доступа к какому-либо реальному счёту.
- **Расчёт ликвидации** — упрощённая модель (отступ ~95% маржи от цены
  входа, без учёта funding rate и комиссий), сделана для обучения принципу
  работы плеча, а не для точного воспроизведения биржевой механики.
- **Хранение данных** — локально на устройстве через Hive. Если удалить
  приложение, демо-счёт сбрасывается.
